A Local Interface to Send Emails with Postmark

This is a REST API built to be used locally on your machine to manage subscribers from your mailing list (MongoDB Collection) and send Broadcast Emails using the Postmark API.

The application receives your subscriber data from MongoDB and parses the JSON information into a table. You can then send an email to each subscriber with a postmark templated email by inputing the id of the template you created in the postmark dashboard.

I chose to use postmark for their features like open and click tracking, custom email templates, spam filtering and bounce handling.

This project was built to manage subscriptions from an application which signs subscribers up and stores them in a MongoDB collection. You can view the sister project here.
