Startup Landing Page With Mailing Service

Google Apps Script + JavaScript + HTML + CSS

www.boundnetwork.com

Bound is a directory for digital creatives to network. I focused on creating a stack which validates the product first then translates the data gathered for product launch.

After looking at existing marketing platforms such as Mailchimp I decided that I wanted to be in full control of monitoring things like CRO (Conversion Rate Optimisation) and SEO (Search Engine Optimisation.) Wanting to also create this project as streamlined and cost-effective as possible, (and have a crack at making my own mailing system) I went ahead to create my own sign-up service.

The website consists of simple copy, minimal design and a Form to submit. The site is hosted on GitHub pages and developed using HTML and CSS (Bootstrap.) The Form operates using Google Apps Script and JavaScript, all evaluations are done on the Google Servers allowing me to host the site on Github for free.

How The Form Works:

The HTML Form has an action of POST to a google web app url where I have used the Google Apps Script language to store data in a google sheet.

Google sheets:

By using Google Apps Script I can choose the type of email content to send to specific users. This is helpful in asking specific groups of users what they would like from the product. The data from google sheets is also exportable to JSON to store in a database if the application requires it in the future.
